# Система Управления Ресторанами и Доставкой

## Introduction

Проект представляет собой **RESTful API** для комплексного управления системой доставки еды, включающей управление ресторанами, меню, корзинами пользователей, заказами, курьерами, платежами и отзывами.

## Contents

- [Used technologies](#used-technologies)
- [Start-up](#start-up)

## Used-technologies
*   **Java 17**
*   **Spring Boot 3.3.5**
*   **Аутентификация и авторизация с использованием Spring Security c JWT**
*   **База данных H2**
*   **Docker-compose**
*   **Swagger (SpringDoc OpenAPI)**

## Start-up

Выполнить команду `docker-compose up` в терминале для развертывания проекта.

## Program-operation

Можно узнать все rest endpoint приложения через сваггер, введя в браузере:
http://localhost:8082/swagger-ui/index.html
